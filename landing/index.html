UppASD

Atomistic spin dynamics and Monte-Carlo simulations of magnetic materials

About UppASD

The UppASD (Uppsala Atomistic Spin Dynamics) software package is a simulation suite to study magnetization dynamics by means of the atomistic version of the Landau–Lifshitz–Gilbert (LLG) equation.

Detailed information about the method can be found in
Atomistic Spin Dynamics: Foundations and Applications, O. Eriksson et al., Oxford University Press (2017) .

Applications

  • Critical temperature determination
  • Magnon spectra simulations
  • Micromagnetic exchange and DMI stiffness calculations
  • Current, field, and temperature-driven dynamics
  • Linear spin-wave theory

Features

  • LLG, Metropolis, and Heat-bath algorithms
  • Generalized Hamiltonian including:
    • bilinear and biquadratic Heisenberg exchange
    • Dzyaloshinskii–Moriya exchange
    • magnetocrystalline anisotropies
  • Spin-lattice simulations
  • Multiscale functionalities
  • Support for arbitrary crystal structures and random alloys
  • Spin-transfer and spin-orbit torques
  • Temperature models for ultrafast demagnetization simulations
  • Temperature gradients
  • Quantum statistics

Resources

Installation

Precompiled binaries are available from the Release page.
For production use, building from source, using cmake, is recommended. 

cmake -S . -B build
cmake --build build

Documentation

  • User manual — installation instructions, theory, and full reference.
  • Tutorials — hands-on walkthroughs and ready-to-run examples.

Source code and community

The source code is hosted on github.com/UppASD/UppASD.

For bug reports and feature requests use the GitHub Issues page. User discussions are encouraged on the GitHub Discussions page.

Graphical User Interface

A Python-based Qt GUI (asd_gui) is included, providing VTK-based visualization and plotting via Matplotlib. 

pip install asd_gui

UppASD outputs can also be visualized using SpinView, a standalone application for graphical analysis of magnetic simulations.

Python package

UppASD is also available as a Python package for embedding in workflows and notebooks. The Python interface is in a preliminary stage and is not feature complete.

pip install --index-url https://test.pypi.org/simple/ \
--extra-index-url https://pypi.org/simple uppasd
